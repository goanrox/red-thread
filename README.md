# 🕵️ Red Thread — Premium Cinematic Mystery Platform

> Built by Xavier Serrao | Prestige Interactive Detective Experience

## What It Does
Red Thread is an immersive murder mystery platform where 
players investigate high-stakes cases, interrogate suspects, 
collect evidence, uncover hidden motives, and make one 
final irreversible accusation.

Designed as a premium cinematic experience, Red Thread 
blends detective gameplay, narrative storytelling, and 
Apple-inspired design into a polished interactive product.

## Features
- 🧩 **The Thornwood Affair** — Fully playable flagship case
- 🕵️ **Suspect Interrogation System** — Question suspects and uncover lies
- 📁 **Evidence Board** — Review clues, witness statements, and discoveries
- 🕰️ **Timeline View** — Reconstruct the sequence of events
- ⚖️ **Final Accusation Flow** — Choose the suspect and face the consequences
- 🏆 **Detective Rank System** — Earn scores, achievements, and progression
- 📝 **Notes & Progress Persistence** — Save clues, suspicions, and case progress
- 🎭 **Branching Endings** — Different outcomes depending on who you accuse
- 📱 **Responsive Experience** — Optimized for desktop, tablet, and mobile
- 🤖 **Future AI Interrogation System** — Planned live suspect conversations powered by AI

## The Problem It Solves
Most mystery games are either too shallow, too scripted, 
or visually outdated. Red Thread creates a more immersive 
and premium experience by combining rich storytelling, 
meaningful player choices, replayability, and elegant UI.

It is built for players who want to feel like a real 
detective rather than simply clicking through a generic game.

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Zustand with persist middleware
- Framer Motion
- Lucide React
- Local persistence with localStorage
- Deployed on Vercel

## Architecture
Built with a modular Next.js architecture using reusable 
components, route-based case flows, persistent local state, 
and a scalable structure for adding future mystery cases, 
AI-powered suspect conversations, and multiplayer features.
